不bb直切主题 : )

入门

如何安装和进入Zoom视频会议 (生肉)

进阶

画笔工具的使用

全屏状态: 顶部”查看选项”的按钮 => 注释

保存: 可以将你的笔记保存为图片

锁定视频对话框

视频框来回切换不舒服怎么办: 把对话框给锁了

需要锁的对话框 => 右上角三个点 => 选定视频

取消同上步骤

举手发言

正常模式: 右上角”参会者” => 有 “静音”和”举手”

全屏模式: 同是”参会者” => “举手”

取消举手键位和举手键位一致

掌声

全屏状态: 表情 => “掌声”和”抬拇指”

高阶玩法

需要电脑带的动才可以, 带不动就一卡一卡的

画廊模式

所谓的全局视角😂

Zoom共有三个视图模式: 演讲者模式、画廊模式和迷你窗口模式

演讲者模式: 在右上角有个按钮, 切换到单人模式

画廊模式: 切换到多人全局模式

迷你窗口模式: 把窗口最小化即可出现你当前选中的视频画面的迷你模式

虚拟背景

进入设置:

​ 未开始会议: 语音设备设置 => 虚拟背景

​ 开始会议: 停止视频 =>更多 => 虚拟背景 =>加号添加背景

美颜

在设置中的”视频”选项卡 => 美颜

结尾

大家一定可以顺利毕业的👩‍🎓
